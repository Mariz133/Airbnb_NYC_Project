# NYC Airbnb Analysis - Milestone 1

This project analyzes Airbnb listings in New York City (2019 dataset from Kaggle).  
It includes:

- Data description and assessment
- Data cleaning
- Single-variable visualizations (price, room type, number of reviews, availability)
- Multi-variable visualizations (price vs reviews, price by room type and neighborhood)

## Dataset

The dataset contains 48,895 listings with 16 columns, including price, room type, neighborhood group, number of reviews, and availability.

## Usage

All code is in `airbnb_analysis.ipynb`.  
Visualizations can be found on the accompanying webpage.
